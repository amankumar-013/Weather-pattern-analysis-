## Project Overview
This project is a web-based dashboard that analyzes daily weather patterns. It uses an embedded dataset to calculate average temperature, humidity, and total rainfall, and visually plots the temperature trends over time.

## How to Run the Project
Because this project is built with web technologies, it requires **zero installation**!

1. Clone or download this repository to your computer.
2. Ensure all three files (`index.html`, `style.css`, and `script.js`) are saved in the same folder.
3. Simply double-click the `index.html` file to open it in any web browser (like Chrome, Edge, or Firefox).
4. The web page will display the data summary and an interactive graph.
